You can also use the API directly:

Create a paste:

curl --data-binary @file.txt /api/v1/paste

Retrieve as plain text:

curl /api/v1/paste/<id>

Retrieve with rendering:

curl /api/v1/paste/<id>.<ext>

Delete a paste:

curl -X DELETE /api/v1/paste/<id>

Bash function for easy pasting:

function paste() {
    local file=${1:-/dev/stdin}
    curl --data-binary @${file} /api/v1/paste
}
